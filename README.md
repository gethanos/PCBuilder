# PC Builder - CITEd.gr VLE

Ένα διαδραστικό εκπαιδευτικό εργαλείο συμβατότητας υπολογιστών

## Ζωντανή Επίδειξη
**Δείτε το ζωντανά εδώ:** https://vle.cited.gr/apps/pcbuilder

## Σύνοψη
Το PC Builder είναι ένα διαδραστικό εργαλείο προσομοίωσης συναρμολόγησης υπολογιστών, σχεδιασμένο για εκπαιδευτικούς σκοπούς. Βοηθά μαθητές και εκπαιδευτικούς να επιλέγουν εξαρτήματα, να ελέγχουν τεχνική συμβατότητα και να αξιολογούν κόστος/απόδοση για διαφορετικούς τύπους χρήσης.

## Χαρακτηριστικά

### Βασικά Χαρακτηριστικά
- **Διαδραστική επιλογή εξαρτημάτων**: Μητρική πλακέτα, CPU, RAM, GPU, αποθήκευση, ψύκτρα, PSU, κουτί
- **Έξυπνη επικύρωση**: Αυτόματος έλεγχος socket, τύπου RAM, διαθεσιμότητας θυρών, συμβατότητας κουτιού
- **Κατηγορίες χρήσης**: Internet, Office, Gaming (μεσαίο/υψηλό) με διαφορετικές προτεραιότητες
- **Πληροφορίες προϊόντων**: Άμεση πρόσβαση σε specifications και κριτικές

### Εκπαιδευτικές Δυνατότητες
- **Σύγκριση επιλογών**: Διερεύνηση εναλλακτικών και συμβιβασμών
- **Ανάλυση κόστους/απόδοσης**: Αξιολόγηση για διαφορετικούς τύπους χρήσης
- **Κριτική σκέψη**: Τεκμηρίωση αποφάσεων και αιτιολόγηση επιλογών
- **Προσομοίωση αναβαθμίσεων**: Σχεδιασμός μελλοντικών βελτιώσεων

## Τεχνολογίες
- **PHP**: Server-side επεξεργασία
- **HTML5/CSS3**: Σύγχρονο responsive design
- **Vanilla JavaScript**: Δυναμική διαχείριση διεπαφής
- **SVG/Canvas**: Οπτικοποίηση και εικονίδια

## Εκπαιδευτική Αξία

### Για Μαθητές
- Κατανόηση τεχνικών προδιαγραφών και συμβατότητας
- Ανάπτυξη δεξιοτήτων κριτικής σκέψης και λήψης αποφάσεων
- Πρακτική εφαρμογή θεωρητικών γνώσεων υλικού

### Για Εκπαιδευτικούς
- Εργαλείο επίδειξης αρχών συναρμολόγησης υπολογιστών
- Βάση για πρακτικές ασκήσεις και προβλήματα
- Υποστήριξη σε μαθήματα πληροφορικής και τεχνολογίας

## Γρήγορη Εκκίνηση


### Απαιτήσεις
- Web server με PHP υποστήριξη
- Modern web browser

## Γρήγορη Εκκίνηση
1. Clone repository: `git clone https://github.com/gethanos/PCBuilder.git`
2. Άνοιγμα: `cd PCBuilder`
3. Ανοίξτε το `index.php` στον browser

## Χρήση

### 1. Ορισμός Χρήσης
Επιλέξτε τον τύπο χρήσης:
- **Internet/Απλή χρήση**: Βασικές ανάγκες περιήγησης
- **Εφαρμογές γραφείου**: Επεξεργασία κειμένου, πίνακες, παρουσιάσεις
- **Gaming (μεσαίο)**: Casual gaming και multimedia
- **Gaming (υψηλό)**: High-end gaming και streaming

### 2. Επιλογή Εξαρτημάτων
- Κάντε κλικ στο "Επιλογή" για κάθε κατηγορία
- Χρησιμοποιήστε φίλτρα για περιορισμό επιλογών
- Επιλέξτε το κατάλληλο εξάρτημα

### 3. Έλεγχος Συμβατότητας
- Πατήστε "Έλεγχος συμβατότητας" για επικύρωση
- Ελέγξτε τα αποτελέσματα για σφάλματα
- Χρησιμοποιήστε τα badges για γρήγορη ένδειξη

### 4. Ανάλυση και Τεκμηρίωση
- Σημειώστε αποφάσεις και αιτιολόγηση
- Συγκρίνετε διαφορετικές διαμορφώσεις
- Σχεδιάστε μελλοντικές αναβαθμίσεις

## Κατηγορίες Εξαρτημάτων

### Βασικά Εξαρτήματα
- **Μητρική Πλακέτα**: Socket, chipset, τύπος RAM
- **Επεξεργαστής (CPU)**: Συμβατότητα socket
- **Μνήμη (RAM)**: Τύπος (DDR4/DDR5), χωρητικότητα
- **Αποθήκευση**: NVMe, SATA SSD, HDD

### Εξαρτήματα Επιδόσεων
- **Κάρτα Γραφικών (GPU)**: Επίδοση, κατανάλωση
- **Τροφοδοτικό (PSU)**: Ισχύς, αποδοτικότητα
- **Ψύκτρα CPU**: Συμβατότητα socket
- **Κουτί (Case)**: Συμβατότητα μεγέθους

### Περιφερειακά
- **Οθόνη**: Ανάλυση, ρυθμός ανανέωσης
- **Πληκτρολόγιο & Ποντίκι**: Τύπος, σύνδεση
- **Λειτουργικό Σύστημα**: Συμβατότητα

## Σενάρια Διδασκαλίας

### Επίπεδο Αρχάριων

### Επίπεδο Ενδιάμεσο
- Ανάλυση κόστους/απόδοσης

- Σχεδιασμός custom solutions
